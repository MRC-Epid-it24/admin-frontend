Export nutrient mapping

This tool will export the list of all foods in the selected locale including references to the food composition tables they are linked to and the corresponding nutrient data from those tables.

The meaning of columns in the output .csv file is as follows:

Intake24 food code — self explanatory.

English description — food description in English.

Local description — food description in the local language associated with the selected locale.

Source locale — locale that was used to resolve the food composition table code. This will usually be the selected locale, however it can be different if the food has no food composition table links defined int the selected locale and the system has to fall back on the data from the prototype locale.

FCT — the ID of the food composition table that this food is linked to, e.g. NDNS_Yr_12.

FCT record ID — the ID of the record in the food composition table that the food is linked to, e.g. if the FCT is NDNS_Yr_12 then this is the NDNS (NDB) food code

The rest is the nutrient data in units per 100g/ml. The available nutrient types and their order is determined by the locale settings and the unit can be found in the header row.
